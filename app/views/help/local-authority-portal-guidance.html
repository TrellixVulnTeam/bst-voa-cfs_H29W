Local authority portal guidance

Billing authorities can use the local authority portal to:

  • add, remove or amend the details of properties on the lists for Council Tax and non-domestic rates
  • add supporting information to submitted reports
  • view and download schedules
  • find out when new schedules are published

Changes to properties are reported to the Valuation Office Agency (VOA) through billing authority reports. These should include details of the property, the change that is being requested, and the reason for making the change.

There are 2 ways to submit reports:

  • uploading multiple reports using either an Excel or XML file
  • submitting individual reports using an online form

Uploading multiple reports

Select the ‘Upload multiple reports’ option on the local authority dashboard to upload multiple reports to the VOA at once.

You will be asked to upload either an XML file or an Excel file containing all the information the VOA requires to process reports.

XML files can be exported using your own software, or you can use an Excel template to add reports.

Download the Excel template for:

  • Council Tax
  • non-domestic rates

There is a limit of 10MB file size per Excel or XML document.

Once the file has been uploaded, the service will check the file for errors. This usually takes less than 30 minutes and you will be sent an email when this is complete. It will also say if any errors were found.

Fixing errors in uploaded files

If there are any errors in the data when you upload either of these files, the service will say that there are data errors, what these are, and how to fix them.

This usually means that required data has been left out, entered incorrectly, or does not meet format requirements.

You can find out if any of the reports that have been submitted through an XML or Excel file contain errors by selecting ‘Fix upload errors’ on the dashboard.

The service will also check the file itself. If the file is corrupted or contains viruses, you will be asked to delete the file, fix it and try uploading it again.

You will have to do this outside of the system. If you receive a file error, you can try:

  • deleting the file and reuploading it
  • download the Excel template again, if you are using it, then add your reports and reupload
  • export the XML file from your system again, then reupload
  • speak with your local IT team and ask them to check for viruses, corruption, or other issues

If issues continue, contact the Valuation Office Agency.

Creating single reports using the portal

Select ‘Create single reports’ on the dashboard to upload reports individually.

You will be asked what change you want to make, followed by a series of questions where you manually enter the information required for the report.

The report will be submitted to the VOA after all questions have been answered.

What to include in the ‘Remarks’ field

Further information that supports the change that is being requested is required for each report. This is known as ‘Remarks’.

There are corresponding fields within the Excel templates or your XML file for remarks, and there is a box that you can enter them in if you upload reports individually.

What you should include as remarks depends on the reason code you enter for that report.

Find out what to include in the remarks field by reason code

Adding supporting documents

After reports have been submitted to the VOA, they become ‘In progress reports’. This means they are waiting for the VOA to process them before adding to the schedules.

Most in progress reports require supporting information to be processed by the VOA. Supporting information includes documents that support the reason the change has been requested.

Supporting information might include:

  • an inspector’s report
  • photos
  • planning documents

They can be added by selecting ‘Submitted reports’ on the dashboard. This shows all reports that are currently in progress from you billing authority. Select ‘Add supporting documents’ to add files to a report.

The VOA can take up to 90 days from receiving the report to decide whether to add it to the schedules or not. The decision is made based on the information provided in the report and the supporting information you upload to the service.

Downloading schedules

Once reports have been processed and accepted by the VOA they will be added to the schedules. You can choose ‘Download schedules’ on the dashboard to search for and save them.

A filter can also be used to help find schedules of a certain type, or by date.
