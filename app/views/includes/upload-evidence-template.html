How to add supporting documents

Each document should be:

  • Select ‘Choose File’, this will open a window where you can navigate to the supporting document you want to upload
  • Select a supporting document – if the report requires multiple documents, they must be added one at a time
  • Add a document description in the box if you want to – this step is optional by might help the Valuation Office Agency understand how the document relates to the report
  • Use the ‘Add’ button to upload the document

The page will then refresh and a confirmation message will say the document has been added. You can then choose another file. All added documents will appear in a list at the bottom of this page. When you have finished adding documents, select ‘Continue to upload’ to upload them to the service.

Document requirements

Each document should be:

  • no more than 10MB in size
  • in JPG, PNG, PDF, MSG, EML, DOC format

Examples of supporting information include:

  • an inspector’s report
  • photos
  • planning documents
